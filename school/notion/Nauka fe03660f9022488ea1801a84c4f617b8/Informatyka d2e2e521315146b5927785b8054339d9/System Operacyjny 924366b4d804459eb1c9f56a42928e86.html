System Operacyjny

System Komputerowy

Warstwowy model, w którym warstwę najniższą tworzą program i procedury bezpośrednio sterujące sprzętem (ang. drivers). Korzystają z nich procedury obsługi zawarte w jądrze (Kernel), które stanowi właściwy system operacyjny. Jądro zawiera mechanizmy systemowe, które należą do trzech grup:

  • zarządzania procesami
  • zarządzania pamięcią
  • zarządzania systemem plików

System komputerowy składa się na:

  • Użytkowników
  • Oprogramowanie użytkowe
  • System Operacyjny
  • Warstwę sprzętową

System Operacyjny

System operacyjny jest środowiskiem, które pośredniczy pomiędzy użytkownikiem komputera a sprzętem komputerowym. Użytkownik końcowy korzysta z różnych aplikacji, na potrzeby których przydzielane są zasoby systemu komputerowego. Przydziałem tym zarządza system operacyjny, dzięki czemu można uzyskać niezależność programów od sprzętu oraz zapewnić odpowiednio wysoki poziom bezpieczeństwa.

Rodzaje systemów operacyjnych

Systemy operacyjny dzielimy ze względu na interfejs na:

  • tekstowe np: DOS UNIS / Linux
  • graficzne np: Windows, UNIX / Linux, MacOS

Rodzaje systemów operacyjnych ze względu na liczbe wykonywanych programów:

  • Jednozadaniowe np: DOS
  • Wielozadaniowe np: Windows, UNIX / Linux, MacOS, Novell NetWare

Rodzaje systemów operacyjnych ze względu na architekturę systemu:

  • Monolityczne - o najprosztszej strukturze i jednozadaniowe - mogą jednocześnie wykonywać tylko jedno zadanie
  • Warstwowe - o hierarchicznej strukturze poleceń systemowych system może już wykonywać w tym samym czasie kilka poleceń,
  • Klient-serwer - rozbudowana struktura, gdzie serwery pełnią nadzór nad podrzędnymi systemami zainstalowanymi na poszczególnych komputerach działających w sieci.

Rodzaje systemów operacyjnych ze względu na zastosowanie:

  • Zastosowania domowe/biurowe
  • stacja robocza w sieci komputerowej
  • serwer w sieci komputerowej (Windows Server)
  • systemy czasu rzeczywistego - są budowane pod kątem szybkości reakcji na zdarzenie zewnętrzne np: w komputerach pokładowych samochodów, samolotów, jednostek pływających).

Budowa systemu operacyjnego

  • Powłoka systemowa (Shell)

    umożliwia użytkownikowi komunikację z systemem operacyjnym np: Eksplorator Windows, wiersz poleceń (cmd.exe) oraz Windows PowerShell

  • Jądro systemu operacyjnego (Kernel)

    Odpowiada za realizację zadań. W nim znajdują się sterowniki urządzeń, pozwalają na komunikację urządzeń z systemem operacyjnym

  • System plików

    W Windows to NTFS, w linuxie EXT4, a w MacOS APFS - Porządkowanie danych, dodatkowe informacje o plikach tzw, atrybuty

Bitowość systemu - Obecnie najbardziej powszechne są systemy 64-bitowe tj. Windows. Długość rejestrów procesora, dla którego projektowany jest system, wyznacza podstawowy format wewnętrznych zmiennych, znaczników i tablic systemowych, długość ta wyznaczana jest bitach. Również sposób zarządzania innymi zasobami, przede wszystkim pamięcią RAM, zależy od długości słowa
procesora.

System plików

Sposób, w jaki komputer organizuje pliki i katalogi nanośniku danych. System plików określa to, jak informacje są zapisywane i odczytywane. System plików definiuje także wielkość klastrów, możliwe do użycia atrybuty plików oraz schemat poprawnych nazw plików i katalogów - ich długość i dopuszczalne znaki w nazwach.
